## 🚀 프로젝트 개요
- **목적**: 전문가의 투자 인사이트(영상, 오디오, 텍스트)를 집약한 학습 플랫폼
- **기술 스택**: Vite + React + Vanilla CSS + Lucide Icons + React-Markdown
- **배포 방식**: GitHub Actions를 통한 자동 배포 (GitHub Pages)

## ✅ 주요 작업 완료 내역

### 1. 기반 시스템 구축 및 최적화
- **Vite 초기화 및 패키지 설치**: React 기반의 빠른 개발 환경 구축 완료
- **GitHub Pages 배포 설정**: `vite.config.js` 수정 및 자동 배포 워크플로우(`.github/workflows/deploy.yml`) 구축 완료
- **README.md 한글화**: 프로젝트 목적과 구조에 맞게 한국어 가이드 작성 완료

### 2. 콘텐츠 관리 시스템 개선
- **데이터 로직 외부화**: 하드코딩된 데이터를 `public/data/posts.json`으로 분리하여 관리 효율성 극대화
- **동적 마크다운 로딩**: `public/docs/` 폴더의 `.md` 파일을 실시간으로 읽어와 렌더링하는 기능 구현
- **복합 콘텐츠 지원**: 하나의 게시물에서 **영상(YouTube/Drive), 오디오(Drive), 텍스트(Markdown)**를 동시에 보여줄 수 있도록 상세 페이지 고도화

### 3. 기능 및 UI/UX 고도화
- **오디오 플레이어 개선**: 구글 드라이브의 다양한 링크 형식(공유용, 직접 링크 등)을 자동으로 인식하는 파싱 로직 적용
- **UI 클린업**: 불필요한 GitHub 버튼 및 수정 버튼을 제거하여 콘텐츠 몰입도 향상
- **유연한 레이아웃**: 썸네일 유무에 따라 카드의 너비가 자동으로 조절되는 디자인 적용 (`thumbnail` 데이터 없이도 깔끔한 텍스트 리포트 지원)

## 📁 주요 폴더 구조
- `src/`: React 소스 코드 및 스타일(`App.jsx`, `App.css`)
- `public/data/`: 모든 게시글 정보가 담긴 `posts.json`
- `public/docs/`: 실제 리포트 내용이 담긴 마크다운(`.md`) 파일들
