# 📈 위브즈 주식 투자고수방 (Stock-Study)

이 프로젝트는 React와 Vite를 사용하여 구축된 **주식 투자 학습 콘텐츠 플랫폼**입니다. 투자 전문가의 인사이트가 담긴 영상, 글, 오디오 브리핑을 효율적으로 학습할 수 있도록 설계되었습니다.

## ✨ 주요 기능

- **카테고리 필터링**: 언제나 데이트, 굿모닝 담샘, 기업분석도감 등 관심 있는 분야별로 콘텐츠를 모아볼 수 있습니다.
- **다양한 미디어 지원**:
  - **영상**: 유튜브 기반의 투자 전략 강의 시청
  - **오디오**: 구글 드라이브 연동을 통한 데일리 음성 브리핑 청취
  - **텍스트**: 마크다운(.md) 기반의 심도 있는 리서치 리포트 열람
- **반응형 웹 디자인**: 데스크톱과 모바일 어디서든 최적화된 글래스모피즘(Glassmorphism) UI를 제공합니다.

## 🛠 기술 스택

- **Frontend**: React (v19), Vite
- **Styling**: Vanilla CSS (Custom Glassmorphism Design)
- **Icons**: Lucide-React
- **Content**: React-Markdown (공식 문서 및 리포트 렌더링)

## 🚀 시작하기

### 설치
```bash
npm install
```

### 개발 서버 실행
```bash
npm run dev
```

### 빌드
```bash
npm run build
```

## 📁 폴더 구조

- `src/`: 애플리케이션 소스 코드 및 스타일
- `public/docs/`: 투자 리포트 및 브리핑 마크다운(.md) 파일 저장소
- `public/assets/`: 이미지 및 정적 자산
